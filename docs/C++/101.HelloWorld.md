传说一切语言的第一个程序都是“HelloWorld！”

如下为C++的：

```C++
#include <iostream>

int main(int argc, char* argv[])
{
	std::cout << "Hello World!" << std::endl;
  return 0;
}
```

## main函数

```c++
返回值类型 函数名(参数类型 参数， ... )

}
```

main函数的返回值类型一般为int类型，返回0表示执行成功。
