## 基本命令
启动：`nginx`

版本：`nginx -v`

关闭：`nginx -s stop`

安全退出: `nginx -s quit`

重新加载配置文件：`nginx -s reload`

检查配置文件语法：`nginx -t`

查看 nginx进程`ps -aux | grep nginx`

执行`nginx`启动后就可以在主机地址上看到Nginx默认页面了。
